# Day 01 - Summer Training Report

# Python Fundamentals and Programming Basics

**Training Domain:** Artificial Intelligence & Machine Learning (AI & ML)

---

## Objective

The objective of today's session was to build a strong foundation in Python programming by understanding its basic syntax, variables, data types, memory management, operators, and decision-making statements. These concepts are essential prerequisites for learning Artificial Intelligence and Machine Learning.

---

## Topics Covered

### 1. Introduction to Python

The session began with an introduction to Python, discussing its features, simplicity, and wide range of applications in software development, automation, data science, Artificial Intelligence, and Machine Learning. The trainer also demonstrated how to set up Python in Visual Studio Code and execute Python programs.

### 2. Python Output Statements

Different methods of displaying output were introduced using the `print()` function. The concept of printing strings, variables, and formatted output using **f-strings** was explained to produce clean and readable output.

### 3. Variables and Memory Management

The trainer explained how variables are created in Python and how Python dynamically assigns memory during program execution. The concepts of object references, memory allocation, and reference sharing were discussed. Functions like `type()` and `id()` were demonstrated to identify the data type and memory address of objects respectively.

### 4. Python Data Types

The session covered Python's core built-in data types, including:

- Numeric Data Types (Integer, Float, Complex)
- String Data Type (Single-line and Multi-line Strings)
- Boolean Data Type (`True` and `False`)
- `None` Type

The difference between **mutable** and **immutable** objects was also explained with suitable examples. The trainer also introduced the concept of **type conversion**, showing how data can be converted from one type to another whenever required.

### 5. Python Operators

Various categories of Python operators were discussed along with their practical applications:

- Arithmetic Operators
- Relational (Comparison) Operators
- Logical Operators
- Assignment Operators
- Bitwise Operators

The purpose and usage of each operator category were demonstrated through simple programming examples.

### 6. Decision Making

The fundamentals of decision-making in Python were introduced using `if`, `elif`, and `else` statements. The trainer explained how conditional statements work together with comparison and logical operators to control the flow of a program. Methods for taking input from the user were also demonstrated.
