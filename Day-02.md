# Day 02 - Summer Training Report

# Python Operators, Loops and Control Flow

**Training Domain:** Artificial Intelligence & Machine Learning (AI & ML)

---

## Objective

The objective of today's session was to strengthen Python programming skills by exploring additional operators, looping constructs, and control flow statements. The session focused on writing efficient programs using iterative structures and understanding how program execution can be controlled under different conditions.

---

## Topics Covered

### 1. Remaining Python Operators

The session began with the continuation of Python operators that were not covered in the previous class.

#### Membership Operators

The trainer introduced Membership Operators and demonstrated how they are used to check whether a value exists within a sequence such as a list, tuple, or string.

The following operators were discussed:

- `in`
- `not in`

The practical implementation of membership operators was performed using both lists and tuples to understand their behavior in different data structures.

---

#### Identity Operators

The concept of object identity was introduced using Identity Operators.

The following operators were covered:

- `is`
- `is not`

The trainer explained how these operators compare the identity (memory reference) of two objects instead of comparing their values.

---

### 2. Python Loops

The next topic focused on loops, which allow repetitive execution of statements without writing the same code multiple times.

#### The `for` Loop

The trainer explained the syntax and working of the `for` loop.

Topics discussed included:

- Iterating through lists, tuples and strings.
- Using the `range()` function.
- Understanding `start`, `stop`, and `step` parameters.
- Printing multiplication tables using loops.
- Iterating over collections.
- Using the `len()` function to determine the length of strings and other collections.

Several practical examples were demonstrated to improve understanding of iteration.

---

#### The `while` Loop

The trainer explained the working of the `while` loop and its use when the number of iterations is not predetermined.

The importance of updating the loop variable to prevent infinite loops was also discussed. The session included the use of **Ctrl + C** to terminate an accidentally running infinite loop.

---

### 3. Loop Control Statements

Different statements used to control the execution of loops were introduced.

#### Break Statement

The `break` statement was demonstrated to terminate a loop immediately when a specific condition is satisfied.

#### Continue Statement

The `continue` statement was used to skip the current iteration and continue with the next iteration of the loop.

#### Pass Statement

The `pass` statement was introduced as a placeholder for future implementation where no immediate action is required.

---

### 4. Loop with Else Clause

The trainer explained the unique `for-else` construct available in Python.

The concept was demonstrated using searching operations where the `else` block executes only when the loop completes normally without encountering a `break` statement.

---

### 5. Control Flow Statements

The session concluded with an overview of Python Control Flow.

The usage of conditional branching using:

- `if`
- `elif`
- `else`

was revised through practical examples to solve different logical problems.
