---

### 5. Types of Arguments in Python

The trainer introduced different methods of passing arguments to functions.

#### Positional Arguments

Arguments are passed according to the order of parameters defined in the function.

#### Keyword Arguments

Arguments are passed using parameter names, allowing them to be supplied in any order.

#### Default Arguments

Functions were created with default parameter values, enabling optional arguments when calling the function.

The trainer demonstrated how default values improve flexibility while reducing unnecessary code.
