# Exploratory Data Analysis (EDA): Bivariate Analysis and Correlation Analysis

**Training Domain:** Artificial Intelligence & Machine Learning (AI & ML) using Python

---

## Objective

The objective of today's session was to perform bivariate analysis and correlation analysis on the House Price dataset. The session focused on understanding relationships between different features, identifying important factors influencing house prices, and interpreting graphical visualizations to gain meaningful insights for Machine Learning.

---

## Topics Covered

### 1. Bivariate Analysis

- Introduction to bivariate analysis.
- Understanding relationships between two variables.
- Importance of feature relationship analysis before model training.

### 2. House Size vs. Price Analysis

- Scatter Plot visualization.
- Relationship between living area (`sqft_living`) and house price.
- Understanding how house size influences price.
- Using the `waterfront` feature as a color category (`hue`) for better visualization.

### 3. Waterfront vs. House Price

- Box Plot visualization.
- Comparing house prices based on waterfront availability.
- Understanding the effect of waterfront locations on property value.
- Interpreting median values and data spread.

### 4. House Condition Analysis

- Bar Plot visualization.
- Comparing average house prices for different house condition ratings.
- Studying the influence of property condition on selling price.

### 5. Correlation Analysis

- Understanding feature correlation.
- Creating a correlation matrix.
- Visualizing relationships using a Correlation Heatmap.
- Interpreting positive, negative, and weak correlations between numerical features.

### 6. Dataset Insights

- Identifying the strongest feature influencing house price.
- Detecting significant outliers requiring preprocessing.
- Understanding feature importance before Machine Learning model development.
- Saving the filtered dataset for the preprocessing stage.
