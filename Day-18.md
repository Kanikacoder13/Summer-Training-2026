# Day 16 - Summer Training Report

# Decision Tree Classification

**Training Domain:** Artificial Intelligence & Machine Learning (AI & ML) using Python

---

## Objective

The objective of today's session was to understand the fundamentals of Decision Tree Classification and learn how decision trees are used to solve classification problems. The session focused on tree structure, impurity measures, model training, visualization, hyperparameter tuning, and evaluating model performance using the Titanic dataset.

---

## Topics Covered

### 1. Introduction to Decision Trees

- Understanding Decision Trees as a supervised Machine Learning algorithm.
- Applications of Decision Trees in classification problems.
- Advantages and limitations of Decision Tree models.
- Real-world use cases of Decision Trees.

### 2. Decision Tree Structure

- Root Node
- Internal Node
- Leaf Node
- Decision Rules
- Branches
- Understanding how data flows through a decision tree.

### 3. Splitting Criteria

- Introduction to Entropy.
- Understanding Gini Impurity.
- Selecting the best feature for splitting.
- Creating decision boundaries based on feature values.

### 4. Building a Decision Tree Model

- Loading the Titanic dataset.
- Selecting input features and target variable.
- Splitting the dataset into training and testing sets.
- Training a Decision Tree Classifier using Scikit-learn.

### 5. Decision Tree Visualization

- Visualizing the trained Decision Tree.
- Understanding split conditions.
- Interpreting nodes and branches.
- Reading prediction paths from the root node to leaf nodes.

### 6. Hyperparameter Tuning

- Understanding overfitting in Decision Trees.
- Introduction to tree regularization.
- Important hyperparameters:
  - `max_depth`
  - `min_samples_split`
  - `min_samples_leaf`
- Finding the optimal tree depth by comparing training and testing accuracy.

### 7. Model Evaluation

- Predicting outcomes using the trained model.
- Evaluating model accuracy.
- Comparing training and testing performance.
- Understanding the importance of selecting an appropriate tree depth.
